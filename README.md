# 🐍 python-study

파이썬 기초, 머신러닝/딥러닝 예제를 실행하고 결과를 시각화로 정리하는 학습 저장소입니다.

---

## 📌 프로젝트 목적
- Python 문법/자료구조/실습 코드 누적
- ML/DL 예제 실행 경험 축적
- 결과를 차트(PNG)로 기록

## 🧱 구성
- `run_python_programming.py` : 파이썬 기초 실행
- `run_all_ml_dl_examples.py` : ML/DL 예제 실행
- `charts/` : 시각화 결과 이미지
- `outputs/`, `python-outputs/` : 실행 결과

## ⚡ 실행 방법
```bash
python run_python_programming.py
python run_all_ml_dl_examples.py
```

## ✅ 결과 확인
- `charts/` 폴더에서 생성된 PNG 확인
